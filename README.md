# Ziaulhaq Rahimi - Banking Professional Portfolio

A premium, world-class personal portfolio website for a banking professional, designed with a corporate banking aesthetic featuring Dark Navy and Gold accents.

## Overview

This portfolio website showcases the professional experience, skills, achievements, and qualifications of Ziaulhaq Rahimi, a Cashier at Afghan United Bank (AUB) - Herat Head Office.

## Tech Stack

- **HTML5** - Semantic, accessible markup
- **CSS3** - Custom design system with CSS variables, Flexbox, Grid, and animations
- **Vanilla JavaScript** - No frameworks, pure JS for interactivity

No Bootstrap, Tailwind, React, Vue, or Angular used. Everything is handwritten from scratch.

## Features

### Design
- **Dark Navy (#0A2342) + Gold (#D4AF37)** premium color scheme
- **Glassmorphism** effects on cards and navigation
- **Smooth animations** and transitions throughout
- **Responsive design** - works perfectly on all devices
- **Dark/Light mode toggle** with persistent preference
- **Professional typography** using Playfair Display + Inter fonts

### Sections
1. **Hero** - Animated typing effect, professional card, stats counters
2. **About** - Professional summary with quick info cards
3. **Experience** - Timeline with detailed responsibilities and additional roles
4. **Achievements** - Key accomplishments with hover effects
5. **Skills** - Animated progress bars and core competencies
6. **Education & Training** - Academic background and professional certifications
7. **Languages** - Proficiency levels with visual indicators
8. **Contact** - Multiple contact methods with interactive cards
9. **Download CV** - Call-to-action section
10. **Footer** - Navigation links and social connections

### Interactivity
- **Loading animation** with progress bar (auto-hides after 2.5s via CSS fallback + JS)
- **Sticky navigation** with scroll-aware background
- **Smooth scrolling** to sections
- **Active nav link** highlighting on scroll
- **Animated counters** for statistics
- **Animated skill bars** on scroll into view
- **Typing animation** for hero name
- **Back-to-top button** with scroll trigger
- **Mobile hamburger menu** with smooth transitions
- **Keyboard navigation** support (Escape to close menu)
- **Touch swipe** support for mobile menu
- **Reduced motion** support for accessibility

### Accessibility
- Semantic HTML5 structure
- ARIA labels and roles
- Keyboard navigation support
- Focus-visible indicators
- Color contrast compliance
- Screen reader friendly

### SEO
- Comprehensive meta tags
- Open Graph tags for social sharing
- Semantic heading hierarchy
- Fast loading with optimized assets
- Mobile-first responsive design
